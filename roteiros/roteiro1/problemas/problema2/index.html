Paridade

Dada uma mensagem S, sua tarefa é escrever um algoritmo que faça a inserção do bit B na mensagem S, de forma que após a inserção a mensagem S tenha um número par de bits de valor 1.

Nível: 1

Introdução à programção

Descrição da solução

Basta ler a mensagem e contar quantos "1's" tem, e assim acrescentar o bit de paridade.

Pseudocódigo

1 -- S: string
2 -- leia S
3 -- inteiro c = quantidade de 1's que a mensagem S tem
4 -- se c for par
5 -- escreva S + '0'
6 -- senao
7 -- escreva S + '1'

Análise

Para todos os casos o tempo de execução é (O(n)) visto que é precisa percorrer a string para contar os 1's.

Implementação em C++
